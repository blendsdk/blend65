### Coding Standards

## IMPORTANT:

There rules are mandetory and must be applied strictly

1. Allways make sure that you DRY (Don't Repeat Yourself)
2. Allways make sure ALL tests pass.
3. Allways create good code comments explaining the code for maintainability
4. Allways create or update JSDoc for functions, types, enums, constants, class properties.
5. Never create private methods or properties in classes. Methods are either public or protected. Use protected for private methods
